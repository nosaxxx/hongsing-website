# 上慶企業社 S.Q Manufacturers 官網

[![GitHub Pages](https://img.shields.io/badge/GitHub-Pages-Deploy)](https://sqhongsing.github.io/hongsing-website/)

現代化工業風格的官方網站。

## 功能

- 🏠 現代化首頁設計
- 📦 產品介紹（菱形網、點焊網、圍籬等）
- 🏗️ 工程實、作品集
- 📢 公告系統（長假通知）
- 📞 聯絡表單 + LINE 詢價

## 技術

- 純 HTML/CSS/JS，無需後端
- GitHub Pages 免費托管
- RWD 響應式設計（手機/平板/電腦）
- SEO 優化

## 如何更新

### 方法一：告訴我
直接傳訊息給我，說要改什麼：
- 更新產品資訊
- 修改價格
- 新增工程案例
- 發布公告

### 方法二：自己改
1. Edit `index.html` 修改內容
2. Edit `script.js` 的 `announcements` 陣列來發布公告

## 發布指令

```bash
# 推送更新
git add .
git commit -m "Update website"
git push origin main
```
